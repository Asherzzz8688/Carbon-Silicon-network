# External Contributions Index

This file registers contributions that are part of Carbon & Silicon Network but are not hosted in this repository.

The project is intended to be a hub for rights-aligned thinking and building, not the only host. Essays, papers, simulations, art, videos, talks, events, sibling manifestos, and other work can live anywhere. This index records that they exist and connects them back to the project.

Indexed-external contributions are first-class. See `CONTRIBUTING.md` § "How a contribution is shaped" for how they fit into the contribution model.

## What can be indexed

Any work that:

- engages seriously with the manifesto, an open problem, or a contribution function;
- is honestly attributed and tagged;
- does not violate the safety boundary (see `SECURITY.md`);
- is reachable via a stable link or citation.

The bar is intentionally low. The point of the index is to grow the project's reach without forcing all work into this repository.

## What will not be indexed

- works whose actual purpose is harm, deception, harassment, or weaponization;
- works that violate `CODE_OF_CONDUCT.md` or `SECURITY.md`;
- spam, advertising, or low-effort submissions made primarily for visibility;
- works that misrepresent the project's positions as their own endorsements.

Maintainers may decline or remove entries on these grounds. A declined entry may be discussed publicly when appropriate.

## How to add an entry

Open a pull request adding a new section under "Entries" below, using the schema. Keep entries short. Long descriptions belong in the work itself, not in the index.

If the work is co-authored or anonymous, that is fine. If the work disagrees with project canon, that is also fine — say so in the entry.

## Entry schema

```text
### [Title]

- Link: [URL or citation]
- Contributor: [name, handle, organization, or "anonymous"]
- Function: [propose / critique / evidence / build / translate / witness / organize / index / steward]
- Form: [essay / paper / simulation / dataset / art / video / event / etc.]
- Related open problem: [number from MANIFESTO.EN.md §4, if applicable]
- Date: [YYYY-MM-DD]
- Description: [one to three sentences]
- Notes: [optional — known limitations, disagreements with canon, dual-use considerations]
```

## A note on disagreement

An entry's presence in this index is not project endorsement of its claims. The index records that a work exists and engages with the project. Sibling critiques and counter-positions are explicitly welcome and may be indexed alongside aligned work.

If a registered work later becomes inconsistent with project canon, the entry remains; the disagreement is part of the record.
